# Mobile Application

React Native mobile app for iOS and Android.

## Features

- Real-time monitoring
- Push notifications
- Manual irrigation control
- Alert management
- Quick actions

## Technology Stack

- React Native
- TypeScript
- Redux
- React Navigation

## Development

```bash
cd mobile-app
npm install

# iOS
npm run ios

# Android
npm run android
```

This component will be implemented in task 17.
